\hypertarget{index_autotoc_md4}{}\doxysection{Usage üí°}\label{index_autotoc_md4}
Conway\textquotesingle{}s Game of Life is a zero-\/player game based on the evolution of a set of cells arranged in a grid. This grid has an initial random arrangement of living and dead cells and changes according to two rules\+:


\begin{DoxyItemize}
\item Any live cell with two or three neighbors survives.
\item Any dead cell with three live neighbors becomes a live cell.
\item All other live cells die in the next generation. Similarly, all other dead cells stay dead.
